# MineBerry OPSurvival RTP Locations
A list of locations where players are RTP'd in MineBerry OPSurvival
This includes (almost) every single location in the RTP command
## How this was done
If you have Immortal rank on MineBerry OPSurvival, this could be easily done.
There is a command called /rtpadmin, which includes information about RTP. I got all locations by simply running /rtpadmin fullinfo and formatting it into a table. (Yes, this was automated of course)
Running /rtpadmin info also gives simple information, which is only how many locations are there in a certain world.
